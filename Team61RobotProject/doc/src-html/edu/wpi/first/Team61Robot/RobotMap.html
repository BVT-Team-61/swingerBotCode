007 * floating around.

011    // following variables to use with your drivetrain subsystem.

016

018    // number and the module. For example you with a rangefinder:
019    // public static final int rangefinderPort = 1;
020    // public static final int rangefinderModule = 1;
021    
022    // Digital outputs
